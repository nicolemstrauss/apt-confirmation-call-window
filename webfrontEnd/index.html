Welcome

We need a few details about your practice locations to configure call scheduling on your behalf. This page will guide you through the process:

  1. Download the pre-formatted Excel template using the button below
  2. Fill in your practice phone numbers, timezones, and preferred call windows (detailed instructions further down)
  3. Save the file as: YourOrganizationName_YYYY-MM-DD.xlsx (e.g. AcmeHealth_2026-05-01.xlsx)
  4. Upload the completed file using the upload area below

The entire process typically takes 5–10 minutes. If you have any questions, reach out to your account representative.

Entering Time Windows

Use 24-hour format: HH:MM-HH:MM

What you wantWhat to enter
One window, morning only08:00-12:00
Two windows08:00-12:00, 13:00-17:00
No calls on this day(leave blank)

Rules:

  • Hours: 00–23, minutes: 00–59
  • Always use two digits (e.g. 08:00, not 8:00)
  • Separate multiple windows with a comma

Available Timezones

  • America/New_York — Eastern
  • America/Chicago — Central
  • America/Denver — Mountain
  • America/Los_Angeles — Pacific
  • America/Phoenix — Arizona (no DST)

✓ Before You Submit

  • Every row has a phone number starting with +
  • A timezone is selected for each row
  • Time windows use 24-hour format (HH:MM-HH:MM)
  • Days with no call windows are left blank
  • alternate_ivr_behavior is TRUE or FALSE for each row
