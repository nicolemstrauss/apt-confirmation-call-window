Welcome!

We need a few details about your practice locations to configure call scheduling on your behalf. This page will guide you through the process:

  1. Fill in your practice phone numbers, timezones, and preferred call windows using the form below
  2. Add more practices if you have multiple locations using the "+ Add Practice" button
  3. Review your entries — the form will highlight any errors in real time
  4. Submit when you're done — your data will be securely uploaded

The entire process typically takes 5–10 minutes. If you have any questions, reach out to your account representative.

Entering Time Windows (Military Time Only)

All call windows must use military time (24-hour format): HH:MM

What you wantWhat to enter
One window, morning only08:00 – 12:00
Two windows (use "+ Add" button)08:00–12:00 and 13:00–17:00
No calls on this dayLeave both fields blank

Rules:

  • Hours: 00–23, minutes: 00–59
  • Always use two digits (e.g. 08:00, not 8:00)
  • Use the "+ Add" button next to a day to add multiple call windows

Available Timezones

  • America/New_York — Eastern
  • America/Chicago — Central
  • America/Denver — Mountain
  • America/Los_Angeles — Pacific
  • America/Phoenix — Arizona (no DST)

✓ Before You Submit

  • Every practice has a phone number starting with +
  • A timezone is selected for each practice
  • Time windows use military time format (HH:MM, e.g. 08:00 not 8:00 AM)
  • Days with no call windows are left blank
  • If Alternate IVR is checked, a description is provided
